# ⚡ CÀI MONGODB - CHỈ 3 BƯỚC

## 🎯 Đây là cách ĐƠN GIẢN NHẤT!

---

## 1️⃣ DOWNLOAD (2 phút)

Vào link này:

```
https://www.mongodb.com/try/download/community
```

Click nút **Download** (file khoảng 300MB)

---

## 2️⃣ CÀI ĐẶT (3 phút)

1. Mở file vừa download
2. Click **Next** → **Next** → **Complete**
3. Click **Next** → **Next** → **Install**
4. Đợi cài xong → Click **Finish**

**Lưu ý**: Để tất cả mặc định, không cần thay đổi gì!

---

## 3️⃣ NÓI VỚI TÔI (1 giây)

Gõ: **"đã cài xong"**

Tôi sẽ:

- ✅ Config file `.env`
- ✅ Restart backend
- ✅ Kết nối MongoDB
- ✅ Bạn có thể đăng nhập ngay!

---

## 🎉 XONG!

Chỉ cần 3 bước:

1. Download
2. Cài đặt (click Next liên tục)
3. Nói "đã cài xong"

**Bắt đầu ngay: https://www.mongodb.com/try/download/community**
